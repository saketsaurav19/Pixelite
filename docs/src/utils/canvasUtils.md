## 🔧 Functions

<code>hexToRgba(hex: string, opacity: number): string</code>

Converts a CSS hex color string (3-digit or 6-digit, with or without `#`) into an `rgba(r, g, b, opacity)` CSS string. Useful for drawing semi-transparent colors on the canvas.

---

<code>findContour(mask: Uint8Array, width: number, height: number, startX: number, startY: number, offset?: { x: number, y: number }): { x, y }[]</code>

Traces the outer boundary of a binary mask region using **Moore Neighborhood Tracing**. Starts near `(startX, startY)`, finds the first boundary pixel (a `1` that has at least one `0` neighbor), then walks clockwise around the shape. Returns a simplified (radial-distance tolerance) array of `{x, y}` points representing the contour. Returns `[]` if no boundary pixel is found.

---

<code>findAllContours(mask: Uint8Array, width: number, height: number, offset?: { x: number, y: number }): { x, y }[][]</code>

Scans the entire mask and extracts contours for **every disconnected region**. For each unvisited filled pixel it calls `findContour`, then flood-fills that region to `0` so future scans skip it. Returns an array of contour paths (each path having more than 3 points).

---

<code>warpPerspective(ctx: CanvasRenderingContext2D, points: { x, y }[], targetWidth: number, targetHeight: number): ImageData</code>

Applies a **perspective (projective) transform** to the image in the given canvas context. Solves for a 3×3 homography matrix from 4 source corner points using Gaussian elimination, then maps every pixel in the target rectangle back to the source image via bilinear interpolation. Returns a new `ImageData` with the warped result.

---

<code>findBestEdgePoint(ctx: CanvasRenderingContext2D | null, x: number, y: number, radius: number): { x: number, y: number }</code>

Snaps a cursor position to the **strongest image edge** within a square neighbourhood of the given `radius`. Computes a Sobel-like gradient magnitude for each pixel in the region and returns the `{x, y}` coordinates of the pixel with the highest gradient. Falls back to the original `{x, y}` if the canvas context is null or an error occurs.
