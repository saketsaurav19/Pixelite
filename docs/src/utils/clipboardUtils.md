## 🔧 Functions

---

<code>copySelectionToClipboard(state: EditorState, merged: boolean = false): Promise&lt;void&gt;</code>

Composites the active layer (or all visible layers when `merged = true`) onto an off-screen canvas, then crops to the current `selectionRect` (or the full document if no selection exists). Saves the resulting PNG data-URL and crop rectangle into the Zustand store via `setClipboardDataUrl` / `setClipboardDataRect`. Does **not** write to the OS clipboard — the store acts as an internal clipboard.

---

<code>cutSelection(state: EditorState): Promise&lt;void&gt;</code>

Copies the current selection to the internal clipboard (calls `copySelectionToClipboard`), then **erases** the selected rectangle from the active layer using the canvas `destination-out` composite operation. Updates the layer's `dataUrl` in the store and records a `'Cut'` history entry. Does nothing if the active layer is locked or has no image data.

---

<code>pasteFromClipboard(state: EditorState, mode: 'center' | 'in_place' | 'into' | 'outside'): Promise&lt;void&gt;</code>

Creates a new layer from the internal clipboard image. The placement depends on `mode`:
- **`center`** — centers the pasted image in the document.
- **`in_place`** — pastes at the exact coordinates it was copied from.
- **`into`** — clips the pasted image to the current selection rectangle (keeps pixels *inside*).
- **`outside`** — clips the pasted image to the inverse of the selection (keeps pixels *outside*).

Records a history entry named `'Paste <mode>'`.
