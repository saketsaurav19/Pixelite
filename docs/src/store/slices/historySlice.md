## 🔧 Slice Actions

---

<code>recordHistory(actionName: string): void</code>

Captures a **deep snapshot** of the current editor state (layers, selection, lighting, vector paths, canvas view, etc.) and appends it to the history stack as a named `HistoryEntry`. Any entries ahead of the current `historyIndex` (from previous undos) are discarded before appending, forming a linear branch.

Also triggers a **2-second debounced auto-save**: generates a JPEG thumbnail from the topmost `.layer-canvas` element and calls `RecentProjectsStorage.saveProjectState` to persist the project to IndexedDB. If the save produces a new project ID, it is written back into the store.

---

<code>undo(): void</code>

Moves `historyIndex` back by one and spreads the snapshot at the new index over the current store state. Does nothing if already at the oldest entry (`historyIndex <= 0`).

---

<code>redo(): void</code>

Moves `historyIndex` forward by one and spreads the snapshot at the new index over the current store state. Does nothing if already at the newest entry.

---

<code>setHistory(history: HistoryEntry[], index: number): void</code>

Directly replaces the history stack and pointer. Used when loading a saved project from `RecentProjectsStorage` to restore a previously recorded undo/redo timeline.
